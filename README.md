# Earnest Eng Journals

A collection of engineering proposals, architecture documents, and living design documents from the Earnest engineering team. Published via GitHub Pages.

## Documents

| Folder | Description |
|---|---|
| `architecture-guardrails/` | Opinionated engineering guardrails — naming, testing, observability, integrations |
| `infra-ai-improvements/` | Automating infrastructure provisioning with AI-assisted tooling |
| `earnest-agents/` | Architecture for the auto-engineering agent system |
| `avp-laps-poc/` | LAPS + AVP proof-of-concept architecture diagram |
| `sdlc-improvements/` | DX initiative — prioritized SDLC improvement tasks |

## Adding a document

1. Create a new folder with a kebab-case name
2. Add a self-contained `index.html` inside it
3. Add a card for it in the root `index.html`
4. Optionally add a `CLAUDE.md` if the doc has its own editorial conventions
5. Bump the version in the root `index.html` footer
6. Open a PR
